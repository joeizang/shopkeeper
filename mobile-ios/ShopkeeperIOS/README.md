# Shopkeeper iOS

SwiftUI iPhone client for Shopkeeper.

Minimum iOS target:
- `iOS 16.0`

Default local backend URL:
- `http://127.0.0.1:5057`

Current implemented shell:
- login and owner registration
- role-aware tab navigation
- dashboard
- inventory list
- sales summary
- credits list
- reports summary
- profile with owner shop/team controls
